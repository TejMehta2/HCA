# SVG import library

Date: 2023/11/29

## Status

Proposed

## Context

We need a carousel package that allows us to easily create a variety of carousels that meet accessibility guidelines while also allowing SSR

## Decision

We decided to use react-slick (https://www.npmjs.com/package/@ant-design/react-slick) as this has a lot of support and fits our needs better than any alternative

## Consequences

There are a few drawbacks to using react-slick, each of which have been recorded in the CarouselCards docs
