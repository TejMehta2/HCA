# SWR

Date: 2024/02/09

## Status

Proposed

## Context

We have several search components on the HCA replatform site (hcamain) which need client-side data fetching.

## Decision

In order to simplify the async data-fetching, we will use the useSWR hook from the library SWR.

## Consequences

The SWR hook provides a dynamic reference to data from our APIs, and handles things like retry, caching, loading state and errors.

Developers can familiarize themselves with SWR at https://swr.vercel.app/
