
### NextJssImage

This abstraction provides a wrapper around the NextJS Image component, and Jss Image component.

If you are in experience editor, you will always see the Jss Image component.

Otherwise you will see the NextJs image component.

The advantage of this is that we can apply responsive srcset, width and height attributes to allow NextJS to optimise our images. This is necessary because Sitecore does not have a built in image cropper.
