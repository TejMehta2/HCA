# ShareCTA documentation

## Additional information

### Share link text mapping

The share link ctaText is mapped to each item by searching the the sharePlatformList array for an item with a corresponding template name of type ShareTemplate.

In order to add future share links with CMSable names, the CMS will need a new entry then this template needs to be updated with a new mapping.

### Themes

The button theme is defined by a prop, the modal content theme is applied automatically inside the component.
