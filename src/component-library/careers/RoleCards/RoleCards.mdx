# RoleCards

## Description

The RoleCards component consists of a subtitle, title, body text, and a number of CardRole card components

On desktop the title and text are sticky to the screen as the user scrolls through the cards on the right

It is intended to be used on the careers homepage

It can be composed like this:

```tsx
<RoleCards
  title={
    <Text variation="display-2">
      Find your role in transforming patient care
    </Text>
  }
  subtitle={<Text variation="subheading-1">Where do you fit in?</Text>}
  bodyText={
    <Text>
      Delivering quality patient care and innovative treatments is achieved
      through diverse people working in diverse roles, which means there&apos;s
      a good chance we&apos;ll have an opportunity to suit you.
    </Text>
  }
>
  <CardRole
    image={
      <Image
        src="/placeholders/children-playing.jpg"
        alt="two children playing"
        width="643"
        height="605"
      />
    }
    icon={<Icons iconName="iconHospitalLarge" />}
    title={
      <Text variation="heading-2" tag="h4">
        Nursing & Front Line Clinical Services
      </Text>
    }
    cta={
      <Button size="small" variation="full">
        <a href="#">
          <span>
            <Icons iconName="iconArrowSmallRight" />
          </span>
        </a>
      </Button>
    }
  />
  <CardRole
    image={
      <Image
        src="/placeholders/children-playing.jpg"
        alt="two children playing"
        width="643"
        height="605"
      />
    }
    icon={<Icons iconName="iconHospitalLarge" />}
    title={
      <Text variation="heading-2" tag="h4">
        Nursing & Front Line Clinical Services
      </Text>
    }
    cta={
      <Button size="small" variation="full">
        <a href="#">
          <span>
            <Icons iconName="iconArrowSmallRight" />
          </span>
        </a>
      </Button>
    }
  />
</RoleCards>
```
