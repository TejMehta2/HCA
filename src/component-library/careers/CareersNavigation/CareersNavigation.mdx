# CareersNavigation

## Description

The CareersNavigation component is a demo of the existing Navigation component, which has been adapted to support a new layout

It is intended to be used in the Careers pages of the site.

It can be composed the same as the [main Navigation component](?path=/docs/site-components-navigation--docs)

The new feature is two new variations for the existing `Main Navigation Links List` template:
- `full` (12 columns wide)
- `simple` (3 columns wide)

Also a new template "spacer" which allows for a single grid column to be skipped.
