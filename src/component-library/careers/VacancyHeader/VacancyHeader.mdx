# VacancyHeader

## Description

The VacancyHeader component is a simple header component for the career vacancy pages.

It shows a title, some basic info (location, clinical/non-clinical, full-time/part-time, job code) and a cta.

Vertical padding is intended to be applied by a wrapper layout component.

This component is intended to be wrapped in a theme.

It can be composed like this:

``` tsx
<VacancyHeaderStories
  title={
    <Text variation={'display-1'}>Staff Nurse - Imaging</Text>
  }
  location={
    <>Lister Hospital – London</>
  }
  clinical={
    <>Clinical</>
  }
  timing={
    <>Full time</>
  }
  vacancyCode={
    <>0041819</>
  }
  cta={
    <Button variation={'full-light'} size="large">
      <a href="#">
        <span>
          Apply <strong>now</strong>
        </span>
        <Icons iconName={'iconArrowRight'} />
      </a>
    </Button>
  }
/>
```
