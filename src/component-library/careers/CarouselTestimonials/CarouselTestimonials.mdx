# CarouselTestimonials

## Description

The CarouselTestimonials component is a carousel which contains an image, thumbnail, body copy, name, and role

The thumbnail should be a smaller version of the main image

Each slide is on a timer and will automatically move to the next slide after 15 seconds. This can be paused by clicking on the pause icon within the thumbnail

The timer will reset each time the active slide is changed

It is intended to be used in the careers homepage
