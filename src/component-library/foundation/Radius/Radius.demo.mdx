# Radius

The radius variables are intended to be imported directly into component scss module files, and assigned to the `border-radius` CSS property.

For example: 

```scss dark
// MyComponent.module.scss
@use '../../foundation/index.module.scss' as *;

.myLargerElement {
  border-radius: $large-radius;
}

.myMediumElement {
  border-radius: $medium-radius;
}

.mySmallElement {
  border-radius: $small-radius;
}
```
