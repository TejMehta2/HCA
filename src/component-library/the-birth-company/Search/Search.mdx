# Search

## Description

The Search component is a simplified version of the Consultant Finder Search component. A slight difference between the two is this version supports subheadings within the dropdown

It takes a list of data and displays in two columns. This can be filtered by typing into the input field

It is intended to be used the first step of The Birth Company booking journey

It can be composed like this:
