# YextResultCardFAQs

## Description

The YextResultCardFAQs component is one of several custom results `CardComponent` used to display Yext search results.

The main component is a display component, but the directory also contains an adaptor component, similar to the Sitecore integration components.

It should be wrapped in a theme.

It can be composed like this:

```tsx
<YextResultCardFAQs title={title}>{content}</YextResultCardFAQs>
```
