# YextResultCardArticles

## Description

The YextResultCardArticles component is one of several custom results `CardComponent` used to display Yext search results. 

The main component is a display component, but the directory also contains an adaptor component, similar to the Sitecore integration components.

It should be wrapped in a theme.

### Visual component

The visual card element can be re-used for other purposes as shown:

``` tsx
<YextResultCardArticles
  image={
    <>
      <Image />
    </>
  }
  title={
    <Text tag="h3" variation={'heading-1'}>
      Card title
    </Text>
  }
  description={
    <Text tag="div" variation={'body-large'}>
      Card description
    </Text>
  }
  cta={
    <Button size={'large'} variation={'full'}>
      <a href={'#'}>Read more</a>
    </Button>
  }
/>
```

### Adaptor component

The adaptor component renders the visual component and maps the props to Yext search result data. 

Its purpose is to integrate the Yext response data with our re-usable visual component, and it is typically consumed by the SDK components as a `CardComponent`:

```tsx
<YextProvider>
  <UniversalResults
    verticalConfigMap={{
      articles: {
        label: 'Articles',
        CardComponent: YextResultCardArticlesAdaptor,
      },
    }}
  />
  <VerticalResults
    CardComponent={YextResultCardArticlesAdaptor}
  />
</YextProvider>
```
