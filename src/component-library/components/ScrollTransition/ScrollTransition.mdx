# ScrollTransition

## Description

The ScrollTransition component is a wrapper for the homepage components to allow the background color to transition as the user scrolls between components.

This component also allows for individual component sections to be animated as they appear on screen for the first time.
This can be achieved by adding `data-animate="s"` to a HTML element. The amount the elements animates depends on the given size, the options are 'xs', 's', 'm', and 'l'

It is intended to be used on the homepage.
