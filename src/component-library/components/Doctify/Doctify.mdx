# Doctify

## Description

The Doctify component is made up of a rating, a reviews count, and a logo. It can be aligned to the left, right or centre. It has theme styling for use on dark or light backgrounds or can be nested in a parent component using a Theme.

It is intended to be used in different parts of the site with one example being the footer.

It can be composed like this:

```tsx
<Doctify
  link={<a href="#"></a>}
  rating={4}
  reviews="13,500 +"
  theme="light"
  logo={{
    dark: (
      <Image
        src="/doctify-dark.png"
        alt="doctify logo"
        width="83"
        height="21"
      />
    ),
    light: (
      <Image
        src="/doctify-light.png"
        alt="doctify logo"
        width="83"
        height="21"
      />
    ),
  }}
></Doctify>
```
