# Accordions

## Description

The `Accordions` component is used to display one or more accordions with an optional header and call to action.

It is intended to be used in within a theme.

It can be composed like this:

```tsx
<Accordions
  header={
    <Text tag="h3" variation="display-3">
      Hip Pain FAQ
    </Text>
  }
  accordions={[
    {
      title: 'How long will I have to wait to book a hip pain appointment?',
      children: (
        <p>
          Eiusmod irure nostrud culpa veniam nisi incididunt nostrud commodo
```
