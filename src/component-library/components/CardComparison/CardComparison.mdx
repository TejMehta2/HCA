# CardComparison

## Description

The CardComparison component is a card which consists of a title, subtitle, cta, pricingVariants, includedPackageItems, and excludedPackageItems. 

It is intended to be used in the CardBlock component

It can be composed like this:

``` tsx
<CardComparison
  cta={<a href="#"><span>Choose Flexible Care GP</span></a>}
  excludedPackageItems={[
    {
      label: 'Annual flu vaccination'
    },
    {
      label: 'Prescription services'
    },
    {
      label: 'Travel & Vaccination advice'
    },
    {
      info: 'additional tooltip info',
      label: 'Annual reAssure I health screen',
      summary: '(worth £326)'
    }
  ]}
  includedPackageItems={[
    {
      label: 'Unlimited GP consultations'
    },
    {
      label: '50% Urgent Care Centre consultations'
    },
    {
      label: 'Specialised women\'s health clinics'
    }
  ]}
  pricingVariants={[
    {
      period: 'per month',
      price: '£22.99'
    },
    {
      discount: '-10%',
      period: 'per year',
      price: '£236'
    }
  ]}
  subtitle={<Text tag="p" variation="body-medium-extra-large">All ages</Text>}
  title={<Text tag="h3" variation="display-5">Flexible Care GP</Text>}
/>
```
