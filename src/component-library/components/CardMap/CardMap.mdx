# CardMap

## Description

The CardMap component consists of a title, address, and 2 CTAs. There are also optional fields for image, distance, and an extra cta.

The extra CTA is for use in the map component and can be used to close the information popup cards. There are a few styling tweaks applied when this card is used within the map

It is intended to be used on any page and should be nested in a theme.

It can be composed like this:

```tsx
<CardMap
  theme="A-HCA-White"
  title={
    <Text tag="h3" variation="heading-2">
      The Harley Street Clinic
    </Text>
  }
  address={
    <Text tag="p" variation="body-large">
      35 Weymouth Street W1G 8BJ London
    </Text>
  }
  ctas={{
    button1: (
      <a href="#">
        <span>
          Learn <strong>more</strong>
        </span>
      </a>
    ),
    button2: (
      <a href="#">
        <span>
          Get <strong>directions</strong>
        </span>
      </a>
    ),
  }}
/>
```
