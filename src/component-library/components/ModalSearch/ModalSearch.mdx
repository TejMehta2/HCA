# ModalSearch

## Description


The ModalSearch component is an instance of the Modals component, specialised for displaying a search bar and suggestion buttons.

The implementation of the open/close behavior is inherited from the Modals component, see the Modals documentation for further implementation details.

It can be composed like this:


``` tsx
const dialogRef = useRef<HTMLDialogElement>()
<button onClick={() => dialogRef?.current?.showModal()}>Open</button>

<ModalCallUs 
  ref={dialogRef}
  searchBar={
    <SearchBar placeholder="Start typing..." />
  }
  suggestions={
    <>
      <Button key={index} size={'small'} variation={'full-light-blue'}>
        <button>
          <Icons iconName="iconSearch" />
          <span>{suggestion}</span>
        </button>
      </Button>
      {/* ... */}
    </>
  }
/>
```
