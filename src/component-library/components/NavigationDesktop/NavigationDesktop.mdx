# NavigationDesktop

## Description

The NavigationDesktop component is the version of the main site navigation and header which appears only on desktop and larger devices.

The navigation shares the same props as its sibling: the NavigationMobile component.

It is only inteded to be rendered in the `site-components/Navigation` component. See there for shared features between desktop and mobile.

### User interaction

The main navigation tabs are visible when the control is hovered or focused.

The primary row of controls e.g. "Service & Treatments", "Tests & Scans"... is a list of radio input fields, to support navigation, without JS event listeners.

The navigation draws which show the secondary and tertiary levels are rendered depending on the checked state of the radios.
