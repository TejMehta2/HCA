# BlogFilterList

## Description

The BlogFilterList component is for displaying the current active filters applied in the blog listing page.
Each can be clicked to remove it and clicking the 'Clear filers' button will remove all of them

It is intended to be used in the blog listing page

It can be composed like this:

```tsx
const filters = [
  'Orthopaedic care',
  'Orthopaedic care',
  'Orthopaedic care',
  'Orthopaedic care',
  'Orthopaedic care',
];

const clearFilters = (index: number) => {
  if (index === undefined) {
    console.log('clear all filters');
    return;
  }
  console.log('clear filter ' + index);
};

<BlogFilterList filters={filters} clearFilters={clearFilters} />;
```
