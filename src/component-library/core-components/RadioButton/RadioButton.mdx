# RadioButton

## Description

The RadioButton component is intended to be used as a child of the `RadioButtons` component. `label`, `value` and `name` are mandatory props. `disabled` and `mode` are optional (with `light` being set as the default).

It can be composed like this:

```tsx
<RadioButtons>
  <RadioButton label="example A" mode="light" name="test" value="example-a" />
  <RadioButton label="example B" mode="light" name="test" value="example-b" />
  <RadioButton label="example C" mode="light" name="test" value="example-c" />
</RadioButtons>
```
