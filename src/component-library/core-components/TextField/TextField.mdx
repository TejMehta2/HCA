# TextField

## Description

The TextField component is one of the form fields which can be used on a page. The default type is 'text', but can also be set to 'email'. There is also the option to add a label, tooltip, and validation

It is intended to be used in the payment form

It can be composed like this:

```tsx
<TextField
  id="input1"
  label="Field label"
  required={true}
  pattern="^[a-zA-Z0-9- ,.:!*#~&/()+%'`@-]{1,50}$"
  errorMessage="Error message"
/>
```
