# RadioButtons

## Description

The RadioButtons component is accepts the `RadioButton` component as children and displays a group of radio buttons.

It is intended to be used in forms

It can be composed like this:

```tsx
<RadioButtons>
  <RadioButton
    key="1"
    label="example A"
    mode="light"
    name="test"
    value="example-a"
  />
  ,
  <RadioButton
    key="2"
    label="example B"
    mode="light"
    name="test"
    value="example-b"
  />,
  <RadioButton
    key="3"
    label="example C"
    mode="light"
    name="test"
    value="example-c"
  />
  ,
</RadioButtons>
```
