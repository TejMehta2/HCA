# Button

The Button component is intended apply styles by wrapping a plain button tag or a plain HTML link (in the `children` prop) which can be returned by wrapping a JSSLink component or NextJS Link component.

The theme is built up of the button color - dark or light, variation - outline or full. There are variations of dark and light for the social buttons which have their own theme. This is controlled by the `theme` prop.

The size prop controls the button padding to show the appropriate button size - small or large

To use an icon include it before or after your link or button tag

e.g.

```
<Button>
  <button>
    button text
    <Icons />
  </button>
</Button>
```

or

```
<Button>
  <button>
    <Icons />
    button text
  </button>
</Button>
```

For loading buttons include the `Loader` component as one of the `Button` components children and set the loading prop on the `Button` component. The contents of the button should be wrapped in a span which conditionally adds the `visibility-hidden` class if the button is loading.

```
  <Button theme="full-dark" size="large" loading={loading}>
    <JssLink field={props.fields.Link}>
      {loading ? <Loader theme="dark"></Loader> : ''}
      <span
        className={loading ? 'visibility-hidden' : ''}
      >
        'test text'
      </span>
    </JssLink>
  </Button>
```
