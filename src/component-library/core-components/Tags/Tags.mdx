# Tags

The Tags component is intended apply styles by wrapping a plain HTML link tag (in the `children`   prop) which can be returned by wrapping a JSSLink component or NextJS Link component.

The theme (comprising of a background and text colour) are controlled by the `theme` prop.

The background of the Tags' parent component is intended to be determined by the parent, e.g. in the case of the `theme="white"`  version, which is displayed on an orange background in the designs.
