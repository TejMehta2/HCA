# CardBlock

## Description

The CardBlock component is design to present a variety of different Card components in a grid format.

It is intended to be used in as page level component.

The `variation` prop dictates the type of layout presented: `side-by-side`, `3-columns` or `4-columns`.

The `gapSize` prop dictates the space between the header (title, cta, copy) and card grid.

It can take the following card components as children:

- BlogCard
- PatientStoriesCard
- DoctorCard
- ContentCard
- LocationCard

It can be composed like this:
