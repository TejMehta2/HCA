# HeaderBlogDetails

## Description

The HeaderBlogDetails component consists of a category tag, publish date, and title

It is intended to be used as the header for individual blog articles

It can be composed like this:

```tsx
<HeaderBlogDetails
  theme="F-HCA-White"
  tag={
    <Tags contentVariation="quote">
      <a href="#">Announcement</a>
    </Tags>
  }
  date={<time dateTime="Sept 7, 2023">Sept 7, 2023</time>}
  title={
    <Text tag="h1" variation="display-1">
      The Harley Street Clinic retain CQC &apos;Outstanding&apos; rating
    </Text>
  }
/>
```
