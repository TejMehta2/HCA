# OurLocations

## Description

The OurLocations component is a graphical representation of the location of various HCA sites around the UK.

The desktop version zooms and pans over the UK map to highlight the pins representing the different locations, while transitioning the location cards in and out like a carousel. The change in location is triggered by the user reaching scroll breakpoints.

The mobile version simply shows a list of the pinned maps with accompanying location info cards, without changing the scroll behavior.

It is intended to be used only on the homepage.

### Usage

The props are constructed such that a CMS admin will be able to create/edit the location cards in order to manage the visible locations.

The expectation is that if a new location is needed, then the content admin will be able to determine the `mapX` &  `mapY` position for the new map pin on the existing map background image, expressed as a decimal from 0 to 1.

See [AdvancedBlockHeader](?path=/docs/components-advancedblockheader--docs) for more info on header props.

It can be composed like this:

      ),
  }

        mapScale: 0.5, // scale/zoom of map while pin visible (1 is full size)
        theme: 'B-HCA-Navy-Blue', // theme of map card

          <CardLocation

              <Text tag="p" variation="heading-2">

              </Text>
            }
            subtitle={
              <Text tag="p" variation={'subheading-2'}>
                Scroll down to explore
              </Text>
            }
            icon={<Icons iconName={'iconArrowDown'} />} // Icon or CTA as per designs

              <a href="#">
                <span>
                  View <strong>all</strong>
                </span>
              </a>
            }
          />
        )
      }

    ]
    // Optional props

  }
/>
