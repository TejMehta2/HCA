# TabsBlock

## Description

The TabsBlock component uses the Tabs component to hide / show different content depending on the selected tab

It uses the react-slick carousel to handle the switching from one tab to another, but does not have any of the touch controls

There is a typescript error showing on the Slider tag related to the ref. It is believed to be a bug with the external library, but the error does not cause any issues with the functionality of the component

It is intended to be used in any page
