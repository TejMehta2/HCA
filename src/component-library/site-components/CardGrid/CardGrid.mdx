# CardGrid

## Description

The CardGrid component is a grid for displaying content cards so that each card is displayed neatly

It is intended to be used as part of a search component, to render the results. 

It should be nested inside a theme.

It can be composed like this:

```tsx
<CardGrid theme="D-HCA-Teal">
  <CardPatientStories
    key={0}
    title={
      <Text variation="heading-1" tag="h4">
        Cardiac care
      </Text>
    }
    bodyCopy={
      <Text variation="body-large" tag="p">
        There are over 1400 at The Portland, each year. Hear new mums sharing
        theirs
      </Text>
    }
    image={
      <Image
        src="/placeholders/multicard.jpg"
        alt="baby crying"
        width="363"
        height="243"
      />
    }
    link={
      <a href="#">
        <span>
          Read the <strong>Story</strong>
        </span>
      </a>
    }
  />
  <CardPatientStories
    key={1}
    title={
      <Text variation="heading-1" tag="h4">
        Cardiac care
      </Text>
    }
    bodyCopy={
      <Text variation="body-large" tag="p">
        There are over 1400 at The Portland, each year. Hear new mums sharing
        theirs
      </Text>
    }
    image={
      <Image
        src="/placeholders/multicard.jpg"
        alt="baby crying"
        width="363"
        height="243"
      />
    }
    link={
      <a href="#">
        <span>
          Read the <strong>Story</strong>
        </span>
      </a>
    }
  />
</CardGrid>
```
