# PaymentFormHeader

## Description

The PaymentFormHeader component is used on the Payment Form instead of the Navigation component.

It can be composed like this:

```tsx
<PaymentFormHeader
    paymentsText="Secure Online Payments"
    contactText="Any questions?"
    phoneNumber= {
      icon: <Icons iconName="iconPhone" />,
      text: '03332 223 133',
      number: '03332223133',
    }
    openingHours= {
      icon: <Icons iconName="iconClock" />,
      text: 'Mon-Fri 9am - 5:30pm',
    }
></PaymentFormHeader>
```
