# TextBlockContainer

## Description

The TextBlockContainer component is a simple wrapper designed to contain a subset of the CMSable content

- ColumnSplitter
- RowSplitter
- [TextBlockHeader](?path=/docs/site-components-textblockheader--docs)
- RichText
- BlogImage
- Video
- CTAs

``` tsx
<TextBlockContainer></TextBlockContainer>
```
