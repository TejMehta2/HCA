# SideScrollingCards

## Description

The SideScrollingCards component consists of a title, body text, link, and Patient Story cards. 
On mobile and tablet these cards are displayed in a simple grid and are stacked vertically. 
However, on desktop these will be displayed horizantally and will give the illusion of scrolling sideways as the user navigates down the page

The sidescrolling effect is achieved by taking the total width of each card and applying that as the height of the wrapper div. 
The inner container div height is set to `height: 100vh;` and `position: sticky; top: 0;`
As the user scrolls, there is a function which will take the scoll position and set it as the `translateX` of the inner div to give the appearance of scrolling sideways

It is intended to be used only on the homepage
