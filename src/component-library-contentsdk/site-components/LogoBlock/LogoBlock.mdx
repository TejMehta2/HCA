# LogoBlock

## Description

The LogoBlock component is a wrapper to present agrid of logos.

It is intended to be used with the AdvancedBlockHeader component. See the docs for AdvancedBlockHeader for correspodning prop info.

It can be composed like this:

``` tsx
<LogoBlock 
  header={
    <AdvancedBlockHeader />
  }
  logos={
    [
      <Image src="/placeholders/hca-logo.svg" alt="" width="131" height="56" />
      // ...
    ]
  }
/>

```
