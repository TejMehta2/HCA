# IconCtaBlock

## Description

The IconCtaBlock component shows three child items, which each have an icon, title, copy and ctas.

It is intended to present common questions a user may ask, then to direct users to different contact channels or information pages via the CTAs.

It can take a different icon for mobile and desktop via the `iconMobile` and `iconDesktop` props, or simply one responsive icon for both via the `icon` prop.
