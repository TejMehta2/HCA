# SearchWrapper

## Description

The SearchWrapper is a layout component which consumes the `PlainHeader`, `CardGrid` and `Pagination` components to to display a results page.

It is intended to be used for blog results or other searches which display results as cards.

It can be composed like this:

```tsx
<SearchWrapper
  header={
    <HeaderPlain
      subheading={<Text variation={'subheading-1'}>our HCA blog</Text>}
      heading={
        <Text variation={'display-2'}>News & articles about healthcare </Text>
      }
      search={<SearchBar />}
      theme={'A-HCA-White'}
    >
      <Text variation="body-large">
        Quis laboris proident sint amet id cillum do dolor in tempor est.
        Exercitation aute sint tempor eu ut aliquip commodo enim nulla et
        laborum et culpa minim. Commodo ex laboris pariatur labore nostrud
        dolore.
      </Text>
    </HeaderPlain>
  }
  searchDetail={
    <Text tag="h2" variation="heading-1">
      45 articles including ‘Cardiac care’
    </Text>
  }
  showing={
    <Text tag="p" variation="body-medium">
      Showing 1-12
    </Text>
  }
>
  <CardGrid theme="A-HCA-White">{/* ... cards */}</CardGrid>
  <Pagination pageCount={1} callback={console.log} />
</SearchWrapper>
```
