# YextResultCardLocations

## Description

The YextResultCardLocations component is one of several custom results `CardComponent` used to display Yext search results.

The main component is a display component, but the directory also contains an adaptor component, similar to the Sitecore integration components.

It is intended to be used alongside an interactive map in YextResultSectionLocations.

It should be wrapped in a theme.
