# YextResultCardConsultants

## Description

The YextResultCardConsultants component is one of several custom results `CardComponent` used to display Yext search results.

The main component is a display component, but the directory also contains an adaptor component, similar to the Sitecore integration components.

It should be wrapped in a theme.

### Adaptor component

The adaptor component renders the visual component and maps the props to Yext search result data.

Its purpose is to integrate the Yext response data with our re-usable visual component, and it is typically consumed by the SDK components as a `CardComponent`:

```tsx
<YextProvider>
  <UniversalResults
    verticalConfigMap={{
      articles: {
        label: 'Consultants',
        CardComponent: YextResultCardConsultantsAdaptor,
      },
    }}
  />
  <VerticalResults CardComponent={YextResultCardConsultantsAdaptor} />
</YextProvider>
```
