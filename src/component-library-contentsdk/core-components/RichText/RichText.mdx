# RichText

## Description

The RichText component is a simple wrapper component to apply RTE styles.

It is intended to wrap HTML and apply styles to the HTML tags within.

It isn't padded, and should be nested into a padded component, like [TextBlockContainer](?path=/docs/site-components-textblockcontainer--docs)
