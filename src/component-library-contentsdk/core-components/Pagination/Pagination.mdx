# Pagination

## Description

The Pagination component can be used to split groups of content onto multiple pages

The page numbers shown are dynamic and are dependant on the current page of the user. The first and last page is always shown along with the previous and next page numbers

It includes a callback function which can be used to fetch the content to be shown on a particular page. This content is always rendered as a card, but any type of card can be used

Page count is a required prop for the initial render

It is consumed on [page searches](?path=/story/site-components-searchwrapper--locations-list), and as part of the [YextCustomPagination yext component](?path=/docs/core-components-pagination--docs)

It can be composed like this:

```tsx
<Pagination
  theme="A-HCA-White"
  pageCount={14}
  callback={(newPage: number) => {
    return getPageContent(newPage);
  }}
/>
```
