# TextButton

## Description

The TextButton component displays a button with no background, text and an optional icon.

It can be composed by wrapping a button, JSS Link or Link tag in the TextButton component. The styles will then be applied to the anchor or button tag nested within the TextButton component.

```tsx
const exampleClick = () => {
  alert('Button was clicked');
};

<TextButton theme="dark">
  <button onClick="exampleClick">
    Text button text
    <Icons iconName="iconArrowSmallRight" />
  </button>
</TextButton>;
```

```tsx
import { Link as JssLink } from '@sitecore-jss/sitecore-jss-nextjs';
<TextButton>
  <JssLink field={props.fields.PromoLink}>
    Text button text
    <Icons iconName="iconArrowSmallRight" />
  </JssLink>
</TextButton>;
```

```tsx
import Link from 'next/link';
<TextButton>
  <Link href="/test">
    Text button text
    <Icons iconName="iconArrowSmallRight" />
  </Link>
</TextButton>;
```
