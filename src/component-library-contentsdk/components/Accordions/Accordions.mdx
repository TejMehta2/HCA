# Accordions

## Description

The `Accordions` component is used to display one or more accordion.

NB: The `Accordion` component is only used when nested in the `Accordions` component which controls the functionality of the Accordions as a group, and should be used even when using a single accordion.

It is intended to be used in within a theme.

It can be composed like this:

```tsx
<Accordions
  accordions={[
    {
      title: 'How long will I have to wait to book a hip pain appointment?',
      children: (
        <p>
```
