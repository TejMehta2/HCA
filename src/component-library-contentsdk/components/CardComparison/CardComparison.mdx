# CardComparison

## Description

The CardComparison component is a card which consists of a title, description, cta, tag, tagVariant, featuresLabel, pricingVariants, includedPackageItems, and excludedPackageItems. 

It is intended to be used in the CardBlock component

It can be composed like this:

``` tsx
<CardComparison
  cta={<a href="#"><span>Get started</span></a>}
  description={<Text tag="p" variation="body-large">GP subscription suitable for all ages</Text>}
  excludedPackageItems={[
    {
      label: 'Travel & Vaccination advice'
    },
    {
      info: 'additional tooltip info',
      label: 'Prescription services'
    },
    {
      label: 'Annual reAssure I health screen',
      summary: 'worth £326'
    },
    {
      label: 'Annual flu vaccination'
    }
  ]}
  featuresLabel={<Text tag="p" variation="body-bold-extra-large">Features</Text>}
  includedPackageItems={[
    {
      label: 'Unlimited GP consultations'
    },
    {
      info: 'additional tooltip info',
      label: '50% discount on initial consultation at HCA UK Urgent Care Centres'
    }
  ]}
  pricingVariants={[
    {
      period: '/ month',
      price: '£23'
    },
    {
      discount: '-10%',
      period: '/ year',
      price: '£236'
    }
  ]}
  tag="Most popular"
  title={<Text tag="h3" variation="display-5">Flexible Care GP</Text>}
/>
```
