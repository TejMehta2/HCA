# CardLocation

### ID: EQTRHCA6

## Description

The CardLocation component is intended to be used within the homepage map component. It consists of a number, title, and CTA

It can be composed like this:

```tsx
<CardLocation
  theme="B-HCA-Navy-Blue"
  amount={
    <Text tag="p" variation="display-1">
      35
    </Text>
  }
  title={
    <Text tag="p" variation="heading-2">
      Locations across the UK
    </Text>
  }
  cta={
    <a href="#">
      <span>
        View <strong>all</strong>
      </span>
    </a>
  }
/>
```
